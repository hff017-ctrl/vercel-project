🏠 삼성래미안부동산 웹사이트 실행 가이드
========================================

✅ 웹사이트가 성공적으로 생성되었습니다!

📁 생성된 파일:
- index.html  (메인 HTML 파일)
- styles.css  (스타일시트)
- script.js   (JavaScript)
- README.md   (상세 설명서)

🚀 실행 방법:

방법 1: 바로 실행 (가장 간단)
-------------------------------
1. "index.html" 파일을 더블클릭
2. 기본 브라우저에서 자동으로 열립니다

방법 2: 브라우저에서 직접 열기
-------------------------------
1. Chrome, Edge, Firefox 등 브라우저 실행
2. Ctrl + O 누르기
3. index.html 파일 선택
4. 열기 클릭

방법 3: 파일 탐색기에서
-------------------------------
1. index.html 파일 우클릭
2. "연결 프로그램" → 원하는 브라우저 선택

✨ 주요 기능:
- 네이버 부동산 스타일 매물 정보
- 유튜브 영상 섹션
- 매물 검색 및 필터링
- 반응형 디자인 (모바일 지원)
- 프리미엄 다크 테마

🎨 커스터마이징:
- script.js 파일에서 매물 데이터 수정 가능
- script.js 파일에서 유튜브 영상 변경 가능
- styles.css 파일에서 색상 변경 가능

📖 자세한 내용은 README.md 파일을 참고하세요!

💡 팁:
- 개발자 도구: F12 키
- 새로고침: F5 키
- 전체화면: F11 키
