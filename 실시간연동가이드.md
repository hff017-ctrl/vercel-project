## 🚀 설치 및 실행 방법

### ✅ 사전 준비

1. **Node.js 설치 확인**
   ```bash
   node --version
   ```
   - 설치 안 되어 있으면: https://nodejs.org 에서 다운로드

### 📦 1단계: 의존성 설치

프로젝트 폴더에서 다음 명령어 실행:

```bash
npm install
```

이 명령어는 다음 패키지들을 자동으로 설치합니다:
- `express` - 웹 서버
- `cors` - CORS 문제 해결
- `axios` - HTTP 요청

### 🖥️ 2단계: 프록시 서버 실행

```bash
npm start
```

또는

```bash
node naver-api-proxy.js
```

**성공 메시지:**

### 🌐 3단계: 웹사이트 열기

서버가 실행 중인 상태에서:

1. **방법 1**: `index.html` 파일을 더블클릭
2. **방법 2**: Live Server 사용 (VS Code)
3. **방법 3**: 브라우저에서 직접 열기

## ✨ 작동 방식

```
네이버 부동산 서버
        ↓
   (데이터 요청)
        ↓
프록시 서버 (localhost:3000)
        ↓
   (JSON 변환)
        ↓
웹사이트 (index.html)
        ↓
   (화면 표시)
```

1. **웹사이트**가 프록시 서버에 매물 요청
2. **프록시 서버**가 네이버 부동산 API 호출
3. 받은 데이터를 **웹사이트 형식으로 변환**
4. 웹사이트에 **실시간 매물 표시**


## 🐛 문제 해결

### 1. "npm을 찾을 수 없습니다" 오류

**해결책**: Node.js를 설치하세요
- https://nodejs.org 에서 다운로드
- 설치 후 터미널을 다시 시작

### 2. 프록시 서버 연결 실패

**증상**: 웹사이트에 "실시간 매물 연동 서버에 연결할 수 없습니다" 메시지

**해결책**:
1. 프록시 서버가 실행 중인지 확인
   ```bash
   npm start
   ```
2. 포트 3000이 사용 중인지 확인
3. 방화벽 설정 확인

### 3. CORS 오류

**해결책**: 프록시 서버를 사용하면 자동으로 해결됩니다.

### 4. 매물이 표시되지 않음

**확인사항**:
1. 프록시 서버 실행 중인가?
2. 콘솔에 오류 메시지가 있는가? (F12 → Console)
3. 네트워크 연결 확인
